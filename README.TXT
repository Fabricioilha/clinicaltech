Tema: 
    Sistema Web
    Site + Sistema de Ordem de Serviço Online

idéia: 
    Um site para divulgação de serviços técnicos com sistema de "OS" integrado para que os clientes possam:
        1 - Entrar em contato (email, telefone, endereço, mídias sociais)
        2 - Solicitar orçamento através do site
        3 - Acompanhar status de manutenção através da área do cliente
        4 - Consultar orçamento
        5 - Aprovar orçamento
    
    O Sistema enviará um email para cada atividade do cliente alertando tal atividade.
